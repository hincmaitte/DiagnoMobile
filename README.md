# Guía de Consumos en Celulares Realme

Este proyecto es una página educativa orientada al aprendizaje y análisis de consumos en celulares, enfocada en el diagnóstico de fallas durante la reparación de dispositivos móviles.

El sitio presenta información técnica y material de apoyo para comprender cómo interpretar los consumos en una fuente de alimentación, identificar comportamientos normales y detectar posibles fallas en la placa del equipo.

Este proyecto fue desarrollado como práctica de desarrollo web utilizando HTML y CSS, y forma parte de una serie de páginas educativas relacionadas con electrónica aplicada a la reparación de celulares.

## Herramientas utilizadas

- HTML
- CSS
- ChatGPT
- Codex
- Claude Code

## Objetivo del proyecto

El objetivo es crear material educativo simple y accesible que ayude a técnicos y estudiantes a comprender mejor el análisis de consumos en celulares durante el proceso de diagnóstico y reparación, esto es una guia super básica.
